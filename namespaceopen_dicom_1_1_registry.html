◆ DictionaryFileFormat

enum openDicom.Registry.DictionaryFileFormat
strong

Available file formats for dictionary files.


Enumerator
BinaryFile 

Dictionary content is stored not human readable. The implemented binary format is not standardized. Use of file extensions .dic is recommended.


PropertyFile 

Dictionary content is stored in form of a property text file, well-known in context of the Java configuration environment. Files of this format mostly have the extension .properties.


XmlFile 

Dictionary content is stored in form of a XML text file.


CsvFile 

Dictionary content is stored in form of a CSV (Comma Seperated Values) text file, well-known in context of database batch jobs.
