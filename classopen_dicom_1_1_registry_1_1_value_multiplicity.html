◆ IsValid()

bool openDicom.Registry.ValueMultiplicity.IsValid ( int  valueCount)

Determines whether a specified count of values is conform with this DICOM VM instance. This method is supposed to be used for validation of real count of DICOM values against the corresponding DICOM data element dictionary entry.

If this VM instance is not registered in the data element dictionary or this instance is undefined, all specified count of values will be valid.

Lower factor of a VM. In case of "1-3", the lower factor is 1. If the DICOM VM is only one number, the lower and upper factors will be equal.


◆ UpperFactor

int openDicom.Registry.ValueMultiplicity.UpperFactor
get

Upper factor of a VM. In case of "1-3", the upper factor is 3. If the DICOM VM is only one number, the lower and upper factors will be equal. In case of use of variables like "2-2n", the upper factor will only be the corresponding multiplier. In this example the upper factor will be 2.
