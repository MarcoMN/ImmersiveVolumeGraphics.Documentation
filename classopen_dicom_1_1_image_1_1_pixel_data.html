◆ ToArray()

object [] openDicom.Image.PixelData.ToArray ( )

Returns the entire DICOM pixel data as array of binary arrays.

If a DICOM pixel data element is not a DICOM sequence of items, an array with a single binary array entry will be returned. Binary arrays are supposed to be of the type byte[], ushort[] or short[].

◆ ToBytesArray()

byte [][] openDicom.Image.PixelData.ToBytesArray ( )

Returns the entire DICOM pixel data as array of byte arrays.

All non-byte arrays are transcoded into byte arrays. If a DICOM pixel data element is not a DICOM sequence of items, an array with a single byte array entry will be returned.
